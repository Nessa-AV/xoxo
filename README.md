# XOXO vs AI

A modern xoxo game written in Python using Tkinter with an AI opponent.

## Features
- Play against AI
- Three difficulty levels:
  - Easy
  - Medium
  - Hard
- Clean and modern GUI
- Score tracking (wins, losses, draws)
- Play as X or O

## AI Algorithms
This project demonstrates classic game AI algorithms:
- Minimax for decision making
- Alpha-Beta pruning to optimize search performance

These algorithms allow the AI to evaluate possible moves and choose the optimal strategy based on the selected difficulty level.

## Technologies
- Python 3
- Tkinter (GUI)
- Custom game logic
- AI opponent
